🧪 Wireshark Hands-on Lab

This page will document my hands-on practice with Wireshark, a powerful network protocol analyzer.

🔍 Lab Overview

  • Objective: Capture and analyze network packets
  • Tool Used: Wireshark (Mac version)
  • Capture Target: Localhost ping, DNS request, HTTP request

🧑‍💻 Step-by-Step Practice

  1. Install and open Wireshark
  2. Choose a network interface (e.g., en0)
  3. Start capture and run a ping to `google.com`
  4. Stop capture and filter by `icmp`
  5. Analyze the packet details
