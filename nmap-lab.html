🔎 Nmap Hands-on Lab

This page documents my hands-on practice using Nmap for port scanning and network reconnaissance.

🔍 TCP SYN, Version & OS Scan

  • Target: scanme.nmap.org
  • Tools Used: Nmap (Mac)
  • Scan Types: Ping Scan, SYN Scan, Version Detection, OS Detection

📝 Observations

I found open ports and services like HTTP (port 80) and SSH (port 22). The version detection gave service banners, and OS detection suggested a Linux-based host.

💡 Security Interpretation

Even with minimal scanning, many details are exposed. An attacker can identify weak services or outdated versions to exploit.

🎯 Lab Objectives

  • Perform basic port scans using Nmap
  • Identify open ports and services
  • Understand TCP vs UDP scans

🧑‍💻 Commands Used

  • nmap localhost – Scan local ports
  • nmap -sS google.com – Stealth SYN scan
  • nmap -sU 127.0.0.1 – UDP scan on localhost

📌 Observations

I observed that port 80 and 443 were open on certain hosts, and learned how stealth scans work.
